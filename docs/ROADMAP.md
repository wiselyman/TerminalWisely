# TerminalWisely Roadmap

## v0.4.0（当前）

- 任务管理器：当前激活页签对应机器的进程列表（本地 / SSH）
- 搜索、排序、确认后结束进程；抽屉打开时自动刷新

## v0.3.0

- 终端内单击文件路径 → 应用内预览面板（文本、Markdown、CSV、图片、PDF）
- 文本类全文搜索（高亮、上/下一条）
- 预览面板宽度持久化

## 后续评估（未承诺）

| 方向 | 说明 |
|------|------|
| 内置 Web 浏览器 | 在应用内打开 URL，需单独安全与导航设计 |
| Sixel 终端内嵌图 | 终端能力限制大，ROI 低 |
| 多文件 Tab 预览 | 同一会话同时打开多个预览 |
| 结构化 ls 输出 | 提升非标准 listing 的路径识别准确率 |
| 任务管理器增强 | 树形进程视图、网络连接详情、批量结束 |
