# TerminalWisely Roadmap

## v0.5.0（当前）

- Find：当前页签工作目录 `find` 文件搜索（SSH / 本地 Unix）
- 服务器资源：CPU/内存/磁盘/网络/系统信息监控（gauge + sparkline）
- 贴边工具栏三工具：任务管理器、Find、服务器资源（抽屉互斥）

## v0.4.0

- 任务管理器：当前激活页签对应机器的进程列表（本地 / SSH）
- 搜索、排序、确认后结束进程；抽屉打开时自动刷新

## v0.3.0

- 终端内单击文件路径 → 应用内预览面板（文本、Markdown、CSV、图片、PDF）
- 文本类全文搜索（高亮、上/下一条）
- 预览面板宽度持久化

## 后续评估（未承诺）

| 方向 | 说明 |
|------|------|
| Find 增强 | 本地 Windows 支持、`-mtime`/`-size` |
| 资源监控增强 | 单网卡明细、GPU、容器 cgroup |
| 终端 scrollback 搜索 | xterm SearchAddon |
| 多文件 Tab 预览 | 同一会话同时打开多个预览 |
| 任务管理器增强 | 树形进程视图、批量结束 |
