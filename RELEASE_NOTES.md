跨平台桌面终端：本地 Shell + SSH/SFTP。

## 更新内容（v0.5.0）

### Find 文件搜索
- 贴边工具栏打开抽屉，在当前页签工作目录执行 **`find`**
- 输入文件名模式、类型、深度；结果点击目录进入或文件预览
- Ctrl/Cmd+F 快速打开

### 服务器资源检测
- 贴边工具栏第三枚图标，查看当前页签机器的 CPU/内存/网络/磁盘/系统信息与登录用户
- CPU/内存 gauge 与 sparkline；系统信息置顶；网络与 CPU/内存 合为一块布局
- 打开后约 2 秒自动刷新

### 连接体验
- 点击连接后立即出现页签（乐观页签），显示「正在连接…」而非黑屏等待
- 会话就绪后、首条终端输出前显示启动遮罩；连接中页签带脉动指示

### 已知限制
- 本地 Windows 暂不支持 find
- 网络速率需第二次采样；三抽屉互斥

## 更新内容（v0.4.0）

### 任务管理器
- 右侧贴边工具栏图标打开抽屉，查看**当前激活页签**对应机器（本地 PTY 或 SSH 远程）的进程
- 表格列：进程名、监听端口、内存、CPU；支持按名称/端口过滤、列头排序与列宽调整
- × 结束进程需二次确认；抽屉打开时约 2 秒自动刷新

### 已知限制
- SSH 端口解析依赖 `ss` / `netstat`，部分最小化系统可能无端口列
- CPU 为采样值，首次打开可能偏低
- 结束系统或其他用户进程可能权限不足
