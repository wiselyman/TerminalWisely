跨平台桌面终端：本地 Shell + SSH/SFTP。

## 更新内容（v0.3.0）

### 文件预览
- 单击终端中的文件路径，在右侧预览面板打开（本地与 SSH 均支持）
- 文本 / Markdown / CSV 支持全文搜索与高亮跳转
- 图片、PDF 在面板内渲染；过大或不支持类型可「系统打开」
- 预览面板宽度可拖拽调整并持久化

### 页签与工作区
- SSH 页签显示远程系统图标（Ubuntu、Linux 等），替代「SSH」文字
- 页签 `~` 一键回到用户目录；`+` 添加快捷目录，彩色文件夹图标一键 `cd`
- 右键快捷目录可编辑路径；删除需二次确认
- 页签宽度按内容自适应，短名称不再占满整行

### 交互与修复
- 单击目录进入、单击文件预览，自动区分类型
- 修复带引号路径（如 `~/'下载'`）点击报错
- Ctrl/Cmd+点击下载、Shift+点击跨服发送保持不变
