跨平台桌面终端：本地 Shell + SSH/SFTP。

## 更新内容（v0.2.0）

### 文件传输
- 跨服务器文件发送（Shift+点击 / Ctrl+拖拽到标签），经本机内存 relay，无需 SCP
- 上传、下载、跨服发送可并行，底部传输面板独立显示进度与取消
- 下载/上传即时反馈，连接阶段也可取消

### 书签与侧栏
- 书签显示操作系统品牌图标，连接后自动识别 OS
- 支持编辑已保存书签（含密码「留空不修改」）
- 侧栏「+」菜单新建 Local / Remote；收起侧栏显示书签图标 rail，悬停显示别名

### 终端与页签
- Ctrl/Cmd+点击下载、拖拽上传与跨服发送交互优化
- 页签支持拖动排序（不与文件拖放冲突）
- 收起侧栏隐藏多余滚动条
