# v0.5.2 — 上传与终端点击修复

## 修复

### SFTP 上传卡住
- 大文件上传约 9% 后停滞、远端文件仍为 0 字节
- 改为读/写流水线，块大小 512KB，降低 SFTP 并发写，超时 60s

### `ll` 输出误点 owner
- `ll` / `ls -l` 长格式行中，`root`、日期等列不再被识别为可点击文件
- 仅最后一列（文件名）可点击，避免 `No such file` 错误

### 任务管理器端口列
- 修复 `[::]:8000` 等 IPv6 监听地址解析
- `ss` 无 pid 时用 `lsof` / `fuser` 回退
- 从命令行 `--port` 补充显示；子进程端口向上传播到父进程
- 纯数字搜索仅匹配端口数组，不再误匹配进程名中的数字
