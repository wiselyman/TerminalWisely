# v0.3.0 — 控制台打开文件与可检索预览

## 新增

- 应用内预览面板：单击 `ls` 输出中的文件路径打开
- 文本类（含 Markdown 源码）全文 Find：高亮、计数、Enter / Shift+Enter 跳转
- Markdown 预览 Tab（渲染 HTML）
- 图片、PDF、CSV 预览；CSV 表格过滤
- 后端 `preview_open` / `preview_close` / `probe_path`：本地直读 + SSH SFTP 缓存，文本 2 MiB 上限
- 预览面板宽度 localStorage 持久化
- 页签快捷目录：`~` 回主目录、彩色文件夹一键 `cd`、右键编辑与确认删除
- SSH 页签显示远程系统图标

## 改进

- 工作区可拖拽分栏（终端 | 预览）
- 页签宽度按内容自适应
- 产品引导、README 与 MIT 许可证更新

## 修复

- 带引号 shell 路径（如 `~/'下载'`）点击预览/下载报错

## 已知限制

- 路径识别仍依赖 `ls` 输出启发式；非标准 listing 可能无法点击识别
- SSH 相对路径依赖 listing 上下文，`remote_cwd` 漂移时解析可能不准
- PDF 渲染依赖系统 WebView 能力，平台差异需自行验证
