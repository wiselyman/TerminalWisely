# v0.4.0 — 任务管理器

## 新增

- **任务管理器**：工作区**右侧贴边工具栏**小图标 → 点击打开右侧抽屉（叠在预览面板之上）
- 数据范围：**当前激活页签**对应机器（本地 PTY 读本机；SSH 读远程）
- 表格：进程名、监听端口、内存、CPU；× 结束进程（二次确认）
- 交互：进程名/端口过滤；列头排序；列宽可拖拽调整
- 刷新：抽屉打开时约 2 秒自动轮询；关闭后停止

## 后端

- `list_processes` / `kill_process` Tauri 命令
- 本地：`sysinfo` + Windows `netstat2` / Unix `ss`/`lsof` 端口映射
- SSH：exec channel 执行 shell 脚本，stdout 输出 JSON 数组

## 已知限制

- SSH 端口解析依赖 `ss`/`netstat`，部分最小化系统可能无端口列
- CPU 为采样值，首次打开可能偏低，第二次刷新后趋于准确
- 结束系统/其他用户进程可能权限不足（SSH 非 root 时常见）
- 端口展示以**监听端口**为主，不展示全部 ESTABLISHED 连接
