跨平台桌面终端：本地 Shell + SSH/SFTP。

## 更新内容（v0.1.0）

### 核心能力
- 本地 PTY 终端与 SSH 远程终端，多标签在同一窗口切换
- 拖拽文件到 SSH 终端，SFTP 上传到当前远程目录
- 在 `ls` 输出中点击目录名，自动 `cd` 并刷新列表
- Ctrl / Cmd + 点击文件路径，下载到本机 `Downloads/TerminalWisely`
- 本地终端拖入文件时，在命令行插入路径

### 连接管理
- SSH 密码 / 私钥连接
- 连接书签保存与一键重连
- 侧栏 Local / Remote 快捷入口

### 发布与平台
- GitHub Actions 自动构建 Windows、macOS、Linux（含 x86_64 与 ARM64）安装包
- 首次公开发布
