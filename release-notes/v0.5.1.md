# v0.5.1 — SSH 重连与页签菜单

## 新增

### SSH 断开后 Enter 重连
- 远程 Shell 断开后，终端提示「按 Enter 重新连接」
- 在页签内按 **Enter** 即可重新认证并恢复 Shell，无需关闭页签再连
- 后端 `reconnect_ssh_session` 命令；断连时发出 `session-disconnected` 事件

### 页签右键菜单
- 在页签栏 **右键** 打开上下文菜单
- **关闭** / **关闭其他** / **关闭左侧** / **关闭右侧**
- 不可用时对应项自动禁用（如最左侧页签无法「关闭左侧」）

## 改进

- 上传权限失败时提示更明确（SFTP 不使用 sudo 写入）
