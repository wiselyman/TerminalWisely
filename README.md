# TerminalWisely

为日常运维准备的跨平台桌面终端：在同一窗口里管理本地 Shell 与 SSH 会话，用拖拽和点击完成上传、下载、预览与目录浏览。

当前版本：**v0.5.2**

## 它能做什么

### 文件与目录

- **拖拽上传**：将文件拖入 SSH 终端窗口，或拖到 SSH 标签上，通过 SFTP 上传到当前远程目录。
- **点击进目录**：在 `ls` 输出中单击目录名，自动 `cd` 并列出内容。
- **文件预览**：单击 `ls` 输出中的文件路径，在右侧预览面板打开（文本、Markdown、CSV、图片、PDF 等），文本类支持全文搜索与高亮跳转。
- **快捷下载**：Ctrl / Cmd + 点击终端中的文件路径，下载到本机 `Downloads/TerminalWisely`。
- **跨服发送**：在 A 会话的 `ls` 输出中，Ctrl/Cmd + 拖路径到 B 的 SSH 标签；或 Shift + 点击选择目标会话。

### 会话与工作区

- **本地 / SSH 多标签**：页签显示远程系统图标；支持指针拖拽排序。
- **页签快捷导航**：`~` 回到用户目录；彩色文件夹图标一键 `cd` 到常用目录（右键编辑路径，删除需二次确认）。
- **连接书签**：保存常用 SSH 连接，显示 OS 图标，一键重连。
- **传输面板**：多任务上传 / 下载进度与独立取消。
- **任务管理器**：右侧贴边工具栏打开，查看当前页签对应机器（本地或 SSH）的进程、端口、内存与 CPU；支持搜索、排序与确认后结束进程。
- **Find**：贴边工具栏打开，在当前页签工作目录执行 `find` 搜索文件；Ctrl/Cmd+F 快速打开。
- **服务器资源**：贴边工具栏第三枚图标，查看 CPU/内存/磁盘/网络与系统信息；打开后自动刷新。
- **本地终端拖入路径**：拖入文件会在命令行插入路径，便于后续命令使用。

首次启动时，主界面会显示功能介绍；创建终端后进入正常工作区。

## 常用操作

| 操作 | 方式 |
|------|------|
| 进入目录 | 单击 `ls` 中的目录名 |
| 预览文件 | 单击 `ls` 中的文件路径 |
| 下载文件 | Ctrl / Cmd + 单击文件路径 |
| 上传到 SSH | 拖文件到 SSH 终端或 SSH 标签 |
| 跨服发送 | Ctrl/Cmd + 拖路径到目标 SSH 标签 |
| 回到 `~` | 单击页签上的主页图标 |
| 快捷目录 | 页签 `+` 添加；单击彩色文件夹进入；右键编辑或删除 |
| 任务管理器 | 单击右侧贴边工具栏图标；打开后自动刷新进程列表 |
| Find 文件搜索 | 单击贴边工具栏放大镜图标，或 Ctrl/Cmd+F |
| 服务器资源 | 单击贴边工具栏第三枚图标（图表） |

## 快速开始

1. 点击侧栏 **本地终端** 按钮，创建本地 Shell。
2. 点击侧栏 **远程 SSH** 按钮，填写信息并连接。
3. 将常用连接保存为 **书签**，下次点击即可重连。
4. 在 SSH 页签用 `+` 添加快捷目录，或单击 `ls` 中的文件体验预览。
