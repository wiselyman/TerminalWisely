# TerminalWisely

为日常运维准备的跨平台桌面终端：在同一窗口里管理本地 Shell 与 SSH 会话，用拖拽和点击完成上传、下载与目录浏览。

## 它能做什么

TerminalWisely 在普通终端之上，重点提供这些能力：

- **拖拽上传**：将文件拖入 SSH 终端窗口，或拖到 SSH 标签上，自动通过 SFTP 上传到对应会话的当前远程目录。
- **点击进目录**：在 `ls` 输出中点击目录名，自动 `cd` 并列出内容。
- **快捷下载**：Ctrl / Cmd + 点击终端中的文件路径，下载到本机 `Downloads/TerminalWisely`。
- **跨服发送**：Ctrl/Cmd + 拖路径到 SSH 标签，或 Shift + 点击选择目标。
- **本地终端拖入路径**：拖入文件会在命令行插入路径，便于后续命令使用。

此外也支持本地 / SSH 多标签、连接书签等常规终端能力。

## 快速开始

1. 点击侧栏 **本地终端** 按钮，创建本地 Shell。
2. 点击侧栏 **远程 SSH** 按钮，填写信息并连接。
3. 将常用连接保存为 **书签**，下次点击即可重连。

首次启动时，主界面会显示功能介绍；创建终端后进入正常工作区。
